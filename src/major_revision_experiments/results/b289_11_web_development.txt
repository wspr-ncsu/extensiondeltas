A Chrome extension that helps user with UI5 application debugging and troubleshooting. Presents current client application state in a clear and readable manner. Allows to inspect all available frames (with UI5) in the application, inspect all JSON and XML models available in current frame. Extension also enables user to edit JSON models and display changes in the inspected window (except JSON models with circular references – read only). Users can also choose which models should be displayed and which shouldn’t. Extends Chrome DevTools by adding a new panel - “Models”.

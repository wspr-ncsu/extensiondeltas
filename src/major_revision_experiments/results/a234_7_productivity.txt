Using stack notifier you can subscribe for set of tags in Stackoverflow and also get real time notification from StackExchange inbox. Can link user profile to get StackExchange notifications. It will show desktop notifications whenever new questions available for selected tags.
When user click on the notification it will directly open the question in a new tab.
